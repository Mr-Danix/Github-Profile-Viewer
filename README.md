# GitHub Viewer Pro

A sleek, modern GitHub profile visualizer that transforms raw developer data into a clean, interactive dashboard. This tool provides a high-level overview of a user's contributions, achievements, and latest projects using a Glassmorphism design aesthetic.

---

## 🌍 Live Demo   

https://mr-danix.github.io/Github-Profile-Viewer/

---

## 🚀 Features

- Real-time Profile Search – Instantly fetch user data and bios using the GitHub REST API.
- Visual Achievement Trophies – Displays user rank and milestones using the github-profile-trophy API.
- Dynamic Repository Cards – Shows the 6 most recently updated repositories with direct links.
- Stats Dashboard – Quick overview of followers, repositories, and other profile details.
- Modern UI/UX – Dark theme with glassmorphism effects, radial gradients, and smooth animations.

---

## 🛠 Tech Stack

- HTML – Semantic layout and structure  
- CSS – Custom variables, glassmorphism design, animations  
- JavaScript – Fetch API and dynamic DOM updates
